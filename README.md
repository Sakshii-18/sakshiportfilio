# Sakshi Premium Portfolio

A visual-first software portfolio built as a modern frontend application with a focused Java enquiry backend.

## Structure

- `frontend/` contains static HTML, CSS, and JavaScript pages.
- `backend/EnquiryServer.java` contains the lightweight enquiry API.
- `backend/sql/schema.sql` creates the MySQL database and enquiries table.
- `assets/videos/` is prepared for project demo videos.

Open `frontend/index.html` in a browser to view the portfolio. Start the Java backend only when testing the contact form storage flow.
